Use this file as a guide for inserting items into the database.

1. Navigage to server/server.js and open in a text editor
2. Uncomment lines 41-97
  -We don't want production server to see these, but we want to use them internally
3. Change line 51 to the directory and file which stores your Hacker News API data
  -This will likely be in the scraper_data folder of the Scraper repository
4. Remove the last line character from the .txt file you are inserting into the database. There should be no empty lines
5. In the terminal, run app.js from the main Scheming-Lion directory

WARNING: This next step will use all of your computer's resources. Do not expect to complete work side-by-side while inserting items.

6. Open a browser. Type 'http://localhost:3000/importData'
